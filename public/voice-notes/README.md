# Client voice notes

Drop real WhatsApp voice exports here so the **Real Transformations** marquee can play them.

## File names (match `src/data/content.ts`)

| File | Client |
|------|--------|
| `ahmed-k.mp3` or `ahmed-k.ogg` | Ahmed K. |
| `sara-r.mp3` or `sara-r.ogg` | Sara R. |
| `usman-m.mp3` or `usman-m.ogg` | Usman M. |
| `fatima-h.mp3` or `fatima-h.ogg` | Fatima H. |
| `naveed-m.mp3` or `naveed-m.ogg` | Naveed M. |

## From WhatsApp

1. Forward or save the voice note from chat.
2. Convert to `.mp3` or `.ogg` if needed (WhatsApp often uses `.opus` / `.ogg`).
3. Rename to the filename above and replace the placeholder in this folder.

MP3 is preferred; OGG is also supported if you update `audioSrc` in `content.ts` to `.ogg`.
